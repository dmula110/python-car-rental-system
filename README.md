# python-car-rental-system

## Overview

This is a basic object oriented programming project developed in Python. 
The system simulates a simple car rental shop where customers can request rentals, the shop manages inventory, and bills are generated based on rental duration.

The project was created to demonstrate fundamental OOP concepts such as classes, object interaction, and program structure.

## Features
- Display available cars in inventory
- Allow customers to request rentals
- Calculate rental cost
- Generate bills
- Update car inventory after rentals

## Project Structure
ClassRentalshop.py – manages the shop inventory and rental operations

ClassCustomer.py – defines customer behaviour and rental requests

MainBit.py – main program that runs the system and connects the classes

## Technologies
Python  
Object-Oriented Programming (OOP)
